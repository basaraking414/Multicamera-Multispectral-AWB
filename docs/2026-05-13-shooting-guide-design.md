# AWB Transformer 数据集拍摄教程

## 概述

本项目是针对 AWB Transformer（三摄+多光谱融合自动白平衡模型）的数据集采集计划。每场景需拍摄 3 张 RAW + 1 组 MCS，覆盖室内外单一光源场景。

## 环境与器材准备

- 手机/相机支持三摄（Tele/Main/Wide），可输出 RAW（DNG）
- MCS 多光谱模组（按项目已有流程采集 MCS .bin）
- X-Rite ColorChecker（或任何灰卡/色卡）
- 三脚架（可选，建议室内使用）
- 拍摄记录表（CSV，详见附录）

## 拍摄流程

### Step 1: 场景选择

| 类别 | 具体子类 | 建议数量 |
|------|---------|:--------:|
| **室外日光** | 晴天直射、阴影处、树荫下 | 20 |
| **室外阴天** | 多云、阴天、雨雾 | 15 |
| **室外日出日落** | 金色时刻、蓝色时刻 | 10 |
| **室内暖光** | 钨丝灯 2700-3000K | 15 |
| **室内冷光** | LED 日光灯 4000-5000K | 15 |
| **室内荧光灯** | 荧光灯 3500-4500K（偏绿） | 10 |
| **特殊** | 单一光源+小窗户透光 | 5 |
| **总计** | | **90** |

### Step 2: 机位设置

1. **观察场景**：确认是单一光源（关窗、关其他灯、关闭屏幕）
2. **放置色卡**：放在主光源直射区域，朝相机略微倾斜，占比 > 5%
3. **架设手机**：色卡在画面中央区域（不要太靠边，Wide 边缘 MCS 置信度低）
4. **示意**：如下

```
   [光源]
      ↓
   [色卡] ← 相机指向这里（三摄基线对齐色卡）
      ↓
   [相机] ← 固定位置，机身不动
```

### Step 3: 参数设置

| 项目 | 操作 |
|------|------|
| **拍摄模式** | **全自动（曝光自动 + 白平衡自动）** |
| **测光模式** | 点测光（Spot Metering） |
| **对焦/测光点** | 色卡中心 |
| **白平衡** | **自动（不影响 RAW 传感器数据）** |
| **RAW 格式** | 确认开启 |
| **AE/AF Lock** | 如果支持，锁定后切换摄像头 |

### Step 4: 三摄拍摄

```
                 ┌─────────────┐
                 │   Tele 拍摄  │ ← 色卡画面占比最大，测光最准
                 └──────┬──────┘
                        │
                 ┌──────▼──────┐
                 │   Main 拍摄  │ ← 参考帧
                 └──────┬──────┘
                        │
                 ┌──────▼──────┐
                 │   Wide 拍摄  │ ← 检查色卡边缘位置
                 └─────────────┘
```

**操作顺序：**

1. 打开相机 → 切换到 **Tele**
2. 对准色卡 → 点击色卡位置对焦/测光
3. 如有 AE Lock 功能 → 锁定（没有就跳过）
4. 按下快门 → 拍摄 Tele RAW
5. 切换到 **Main** → 自动测光色卡 → 拍摄 Main RAW
6. 切换到 **Wide** → 自动测光色卡 → 拍摄 Wide RAW
7. 记录场景信息到拍摄表
8. **切换机位以改变色卡位置** → 进入下一个场景九宫格位置

> 提示：整个过程不需要移动身体位置，只切换摄像头。保持 10 秒内完成，防止光源变化。

### Step 5: 多位置覆盖

**同一场景下，变换色卡在画面中的位置，增加空间多样性：**

```
┌─────────┬─────────┬─────────┐
│  ① 左上  │  ② 上中  │  ③ 右上  │
│         │         │         │
├─────────┼─────────┼─────────┤
│  ④ 左中  │  ⑤ 中央  │  ⑥ 右中  │
│         │   (主位) │         │
├─────────┼─────────┼─────────┤
│  ⑦ 左下  │  ⑧ 下中  │  ⑨ 右下  │
│         │         │         │
└─────────┴─────────┴─────────┘
```

每格拍 1 个场景（3 张 RAW），共 9 场景 / 光源。

### Step 6: 质量检查

每拍完 3-5 场景后快速检查：

| 检查项 | 标准 |
|--------|------|
| RAW 文件可读 | 用手机预览 DNG 是否正常 |
| 色卡可见 | 三摄画面中是否都有色卡 |
| 不过曝 | 色卡区域无 clipped highlight |
| 不欠曝 | 暗部细节可见（RAW 可提亮 2-3 EV） |
| 文件完整 | 确认 MCS.bin 同步采集 |
| 场景记录 | CSV 表已填写完整 |

## 场景记录表字段

| 字段 | 示例 | 说明 |
|------|------|------|
| scene_id | S001 | 场景编号 |
| date | 2026-05-13 | 拍摄日期 |
| time | 14:30 | 拍摄时间 |
| location_desc | 室内办公室 | 场景描述 |
| light_source | 钨丝灯 | 光源类型 |
| color_temp | 2800K | 估算色温 |
| is_hdr | false | 是否包含强逆光 |
| color_card_pos | 中央 | 色卡位置 |
| tele_focal | 80mm | Tele 等效焦距 |
| main_focal | 28mm | Main 等效焦距 |
| wide_focal | 14mm | Wide 等效焦距 |
| tele_iso | 100 | Tele ISO |
| main_iso | 100 | Main ISO |
| wide_iso | 100 | Wide ISO |
| notes | 关窗拉了窗帘 | 备注 |

## 文件组织

拍摄完成后，按项目要求组织：

```
data/
├── McsBin/          ← MCS .bin 文件（按已有流程采集）
│   ├── S001_tele.bin
│   ├── S001_main.bin
│   ├── S001_wide.bin
│   └── ...
├── image/           ← JPG/DNG
│   ├── S001_tele.dng / .jpg
│   ├── S001_main.dng / .jpg
│   ├── S001_wide.dng / .jpg
│   └── ...
└── scene_log.csv    ← 场景记录表
```

后续按现有预处理流程执行：
```
1. demo_ImageMatchBindata.py  → 匹配 JPG/DNG/MCS.bin
2. dng_decoder.py             → DNG → raw NPZ（不含 GT）
3. gt_extractor.py            → raw NPZ → image_processed NPZ（添加 GT + white_patch）
4. demo_readMcsRaw.py         → MCS.bin → MCS.npy（9 通道）
```

## 不同场景速查表

| 场景 | ISO | AE Lock | 注意 |
|------|:---:|:-------:|------|
| 室外晴天 | Auto | 建议锁 | 避开正午，阴影处色温高 |
| 室外阴天 | Auto | 可锁可不锁 | 光线均匀，最简单 |
| 日落 | Auto | 建议锁 | 色温变化快，三摄要快速拍完 |
| 室内暖光 | Auto | 建议锁 | 关窗关屏幕，单一光源 |
| 室内冷光 | Auto | 建议锁 | 确认无其他光源混入 |
| 荧光灯 | Auto | 建议锁 | 注意频闪，快门别太快 |

## 附录: CSV 记录模板

```csv
scene_id,date,time,location_desc,light_source,color_temp,color_card_pos,notes
S001,2026-05-13,10:30,室外公园,阳光,5500K,中央,
S002,2026-05-13,10:35,室外树荫,阴影,7000K,中央,
```

建议用手机备忘录或 Google Sheets 记录，拍摄后导出 CSV。

## 检查清单（拍摄前过一遍）

- [ ] 手机 RAW 格式已开启
- [ ] 白平衡设为自动（不影响 RAW 传感器数据）
- [ ] 测光模式为点测光
- [ ] MCS 模组已连接并准备就绪
- [ ] 色卡已带（确认色卡无污损）
- [ ] 拍摄记录表已准备
- [ ] 场景为单一光源（关窗/关其他灯/关屏幕）
- [ ] 三摄焦距确认不同（tele > main > wide）
- [ ] 拍完后检查三张 RAW 都正常曝光
